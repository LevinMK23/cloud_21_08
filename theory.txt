- Как организовать клиент-серверное взаимодействие?
	Нужно создать сервер, который откроет сокет для входящих соединений. Требуется указать номер порта, который он будет прослушивать. Клиент должен знать IP адрес сервера и номер порта. При успешном подключении клиента к серверу создаются потоки ввода-вывода текстовые либо байтовые, по которым сервер и клиент обмениваются информацией.
- Как и в каком виде передавать файлы?
	Для передачи файлов используется байтовый поток ввода-вывода.
- Как пересылать большие файлы?
	Для передачи больших файлов используется буфер, чтобы не передавать каждый байт по отдельности. Это заметно сокращает время передачи.
- Как пересылать служебные команды?
	Для передачи служебных команд надо использовать поток ввода-вывода текстовой информации. Принимающая строна парсит передаваемую строку и разбирает её на команды и параметры к ним.
- Что хранить в базе данных?
	В базе данных целесообразно хранить имена и пароли пользователей.
- Как передавать структуру каталогов/файлов?
	Метод list() класса File возвращает список файлов и подкаталогов в указанном каталоге. Возвращает массив строк. Этот массив строк требуется передать по сети запрашивающей стороне.
- Какую библиотеку использовать для сетевого взаимодействия: java.io, java.nio, Netty?
	В интернете пишут, что Netty лучше всех.
